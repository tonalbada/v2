# v2

Kort HTML-test.

## Utveckling

Installera beroenden och kör testerna:

```
npm install
npm test
```

## Deployment

1. Se till att koden är pushad till GitHub.
2. Logga in på [Vercel](https://vercel.com/) och välj "Import Project".
3. Peka på det här GitHub-repot och bekräfta projektroten.
4. Vercel bygger och deployar `index.html` automatiskt utan extra build-steg eller miljövariabler.
